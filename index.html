Welcome!

This website is a resource built for the 2024 Field Genomics Workshop in Ecuador! We have a worked bioinformatic example using Nanopore Adaptive Sequencing Data under the Worked Example tab. We also have all of our code for the analysis of the data we generate during this workshop in the Real Time Example tab. Lastly, there are helpful resources on all packages used in the Resources Tab.
