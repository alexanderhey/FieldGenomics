Suggested for Windows Users: Install Linux on Windows

Note: If you’re using Windows, it’s highly recommended to set up a Linux environment (such as Ubuntu) through the Windows Subsystem for Linux (WSL). Follow these steps to get started.

Step 1: Open Command Prompt and Run WSL Installation

Open your Command Prompt application and run the following command to install WSL:

wsl --install

Step 2: Install Ubuntu and Set Up Your User

Follow the instructions to install Ubuntu, then set up a username and password for your new Ubuntu environment. You’ll be required to restart your computer during this process.

If you need more details or want to download Ubuntu directly, visit the official Ubuntu website.

Step 3: Prepare Your Ubuntu Terminal

Once your Ubuntu terminal is ready, run the following commands to test and install necessary tools for future steps:

sudo add-apt-repository universe
sudo apt update
sudo apt install git-all  
sudo apt install bzip2
sudo apt install build-essential
sudo apt install zlib1g-dev
sudo apt install libncurses5-dev libncursesw5-dev
sudo apt install liblzma-dev
sudo apt install libcurl4-openssl-dev
sudo apt install gobjc gfortran gnat
sudo apt install manpages-dev

Step 4: Create Directories for Your Field Genomics Programs

Before installing programs, create a main directory and a subdirectory for software installations. Run the following commands:

mkdir FieldGenomics
cd FieldGenomics
mkdir software

You are now ready to start installing the necessary programs.

Description: This are the programs you may need if you wish to replicate the analysis performed in the field on the data we generate with the Nanopore run. The participants do not need Dorado for the Worked Example they will be performing on their computers.

Dorado

Description: Dorado is a basecalling and alignment tool used for processing Oxford Nanopore sequencing data.

Website: Dorado GitHub

Installation:

# For Linux (generic)
  wget https://github.com/nanoporetech/dorado/releases/download/v0.7.3/dorado-0.7.3-linux-x64.tar.gz
  tar -xvzf dorado-0.7.3-linux-x64.tar.gz
  cd dorado-0.7.3-linux-x64/bin/
  export PATH=$PATH:$PWD

  # For MacOS (with Homebrew)
  brew install dorado

  # For MacOS (without Homebrew)
  wget https://github.com/nanoporetech/dorado/releases/download/v0.7.3/dorado-0.7.3-macos-x64.tar.gz
  tar -xzvf dorado-0.7.3-macos-x64.tar.gz
  cd dorado-0.7.3-macos-x64/bin/
  export PATH=$PATH:$PWD

  # For Windows (using WSL)
  wget https://github.com/nanoporetech/dorado/releases/download/v0.7.3/dorado-0.7.3-linux-x64.tar.gz
  tar -xvzf dorado-0.7.3-linux-x64.tar.gz
  cd dorado-0.7.3-linux-x64/bin/
  export PATH=$PATH:$PWD

Samtools

Description: Samtools is a suite of programs for interacting with high-throughput sequencing data. It is used for extracting unaligned reads and converting BAM files to FASTA.

Website: Samtools Website

Installation:

  # For Linux (generic)
  wget https://github.com/samtools/samtools/releases/download/1.17/samtools-1.17.tar.bz2
  tar -xvjf samtools-1.17.tar.bz2
  cd samtools-1.17
  make
  export PATH=$PATH:$PWD

  # For MacOS (with Homebrew)
  brew install samtools

  # For MacOS (without Homebrew)
  wget https://github.com/samtools/samtools/releases/download/1.17/samtools-1.17-macos.tar.bz2

BLAST+

Description: BLAST+ is a suite of tools to perform similarity searches against nucleotide and protein databases. We also used blastdb from this. You can either download the latest verseion of BLAST here or use the one we provided.

Website: BLAST+ NCBI

Installation:

  # For Linux (generic)
  wget ftp://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/2.14.0/ncbi-blast-2.14.0+-x64-linux.tar.gz
  tar -xzvf ncbi-blast-2.14.0+-x64-linux.tar.gz
  export PATH=$PATH:$(pwd)/ncbi-blast-2.14.0+/bin

  # For MacOS (with Homebrew)
  brew install blast

  # For MacOS (without Homebrew)
  wget ftp://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/2.14.0/ncbi-blast-2.14.0+-x64-macosx.tar.gz
  tar -xzvf ncbi-blast-2.14.0+-x64-macosx.tar.gz
